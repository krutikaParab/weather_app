# Weather App

A simple weather application that displays current weather conditions and forecasts.

## Features

- Current weather display
- Temperature and conditions
- Location-based weather data
- User-friendly interface

## Installation

1. Clone the repository
    ```bash
    git clone https://github.com/yourusername/weather-app.git
    cd weather-app
    ```
2. Install dependencies
    ```bash
    pip install -r requirements.txt
    ```
3. Configure API keys if needed

    Configure API keys from https://openweathermap.org/api

4. Run the application
    ```
    python app.py
    ```

## Usage

Start the application and enter your location to view the current weather.
